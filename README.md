# Sistema integrado fronterizo: Aduana CAB

AduanaCAB es un prototipo de sistema web orientado a la gestión y control de trámites fronterizos, permitiendo la interacción entre turistas, funcionarios y administradores dentro de una plataforma centralizada.

El objetivo del proyecto es optimizar el proceso de cruce fronterizo mediante la digitalización de formularios, validaciones documentales, seguimiento de trámites e integración con organismos externos.

## Objetivos del Sistema

- Digitalizar los trámites aduaneros.
- Gestionar declaraciones SAG.
- Registrar información de vehículos, menores y mascotas.
- Permitir seguimiento y trazabilidad de trámites.
- Facilitar la validación documental por funcionarios.
- Integrar organismos externos como PDI, SAG y Aduana Argentina.
- Administrar usuarios, roles y auditorías del sistema.

---

## Roles del Sistema

### Turista

Permite:

- Registro de cuenta.
- Inicio de sesión.
- Reserva de turno digital.
- Completar formularios aduaneros.
- Declaración SAG.
- Registro de vehículos.
- Registro de menores.
- Registro de mascotas.
- Seguimiento de trámites.
- Consulta de trazabilidad.

### Funcionario

Permite:

- Consulta de pasajeros.
- Validación documental.
- Gestión SAG.
- Gestión migratoria.
- Generación de reportes.
- Revisión de antecedentes del trámite.

### Administrador

Permite:

- Administración de usuarios.
- Administración de roles.
- Auditoría del sistema.
- Configuración de parámetros.
- Gestión de integraciones.
